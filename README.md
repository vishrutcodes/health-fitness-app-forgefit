## 🚀 About The Project

ForgeFit was born from a simple idea: **fitness guidance should be intelligent, interactive, and accessible to everyone.** Traditional fitness apps are either too generic (one-size-fits-all templates) or too expensive. 

ForgeFit solves this by integrating **Artificial Intelligence** across the entire platform. Using Meta's latest **LLaMA 3.3 70B** model and **Llama 4 Scout Vision** — running on Groq's ultra-fast inference LPU — ForgeFit provides instant, personalized, scientifically-backed coaching.

### What Makes ForgeFit Different?
- **True AI Personalization:** Not templates. Real, dynamic AI responses parsed into beautiful react components.
- **Flawless Mathematical Precision:** The Diet Architect features a custom backend *Corrector Engine* that locks AI hallucinations to exactly match user input macros, resulting in **Zero Drift** generation.
- **Premium Glassmorphism UI:** Framer motion animations, dark-themed frosted glass panels, and stunning Recharts visualizations.
- **100% Free & Open Source.**

## 🔥 Master Features Spotlight

### 1. 🍽️ Diet Architect — AI Meal Plan Generator

The Diet Architect went through massive engineering upgrades to ensure it operates like a true clinical dietitian, free of traditional LLM mathematical hallucinations.

- **Strict Macro Targeting:** Input calories, protein, carbs, and fats. The backend mathematically scales the proportions generated by the AI so that the sum of all meals **matches your input macros to the exact gram.** 
- **Zero-Drift Swapping:** Hate a drafted meal? Click "Swap". The `/api/ai/diet/swap` endpoint generates a completely new dish while strictly maintaining the *exact* target macros of the replaced meal. Your daily total will never drift by a single decimal point.
- **Step-by-Step Recipes:** Every AI-generated meal renders a beautiful recipe card containing the required ingredients and punchy, 2-step preparation instructions.
- **Preferences & Allergies:** Fully handles edge cases like "Vegan", "No Nuts", or "High Volume" foods.

### 2. 🗺️ Fitness Roadmap — AI Training Periodization

ForgeFit generates multi-phase, periodized training protocols that take you from point A to point B safely and scientifically.

- **Phase Generation:** Instantly creates 3-5 macrocycles (e.g., Foundation Phase, Hypertrophy Phase, Peaking Phase).
- **Milestones & Progressive Overload:** The AI dictates exactly how volume and intensity should scale week over week.
- **Workout Schedules:** Full weekly breakdowns matching your available equipment (Bodyweight, Dumbbells, Full Gym).

### 3. 🧠 AI Form Analyzer — ML Vision Exercise Form Coach

The most technically sophisticated feature in ForgeFit. The Form Analyzer combines cloud AI with edge computer vision.

1. **Client-Side MediaPipe:** As the user uploads a video, MediaPipe runs in the browser via WebAssembly to draw a pose skeleton and compute dynamic joint angles (knees, hips, elbows).
2. **Cloud-Side Groq Vision:** The app extracts three base64 keyframes from the video and flashes them to Groq's **Llama 4 Scout Vision** model.
3. **The Result:** The model classifies the exact exercise being performed and returns a score out of 10, along with AI-generated form corrections based on what it actually saw in the video, not from hardcoded rules.

## 📊 Additional Tools & Dashboards

- **Body Progress Tracker (`/progress`):** Log body weight, body fat %, and tape measurements. Visualized with interactive, dark-themed Recharts. Stored securely via Supabase.
- **Personal Records Tracker (`/prs`):** Log your 1RMs on major compound lifts and track strength progression over time. 
- **AI Coach:** A floating conversational widget available on every page, remembering chat context.
- **Exercise Guide + Voice Coach:** Search an encyclopedia of exercises and click the "Voice Guide" button to have the Web Speech API read you step-by-step form cues while performing the lift.
- **5-in-1 Toolkit:** Fully functional BMI, 1RM, Body Fat %, Water Intake, and Sleep cycle calculators.
- **Workout Timer:** Dual-mode precision countdown application supporting standard Rest intervals and Tabata/HIIT phases (Work/Rest/Rounds).
- **Ultimate Fitness Masterclasses:** 10 curated essays covering science-backed periodization, nutrition timing, and biomechanics.

## 🚀 Getting Started

### Prerequisites

| Requirement | Version | Purpose |
|-------------|---------|---------|
| Node.js | ≥ 18.x | JavaScript runtime |
| npm | ≥ 9.x | Package manager |
| Groq API Key | — | Required for AI features ([Get free key](https://console.groq.com)) |

### Installation 

```bash
# Clone the repository
git clone https://github.com/vishrutcodes/health-fitness-app-forgefit.git
cd health-fitness-app-forgefit

# Install dependencies
npm install

# Create environment variables
cat > .env.local << EOF
GROQ_API_KEY=your_groq_api_key_here
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
EOF

# Start development server
npm run dev
```

### Supabase Database Setup

1. Go to your **Supabase Dashboard** → **SQL Editor**
2. Create a **New Query** and paste the contents of `supabase_schema.sql` found in the root.
3. Click **Run** to set up the authentication rules and progress tables.
